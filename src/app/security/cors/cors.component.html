O Compartilhamento de Recursos de Origem Cruzada (CORS) é uma maneira de permitir solicitações HTTP recebidas de diferentes domínios.

É muito comum em aplicativos AJAX, em que o navegador bloqueia todas as solicitações entre domínios, caso o servidor não as autorize.
